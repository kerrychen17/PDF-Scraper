# Banks and Climate Engagement Research

This project analyzes how major U.S. banks respond to climate-related regulation through their public comment letters and related disclosures.

## Overview

The goal of this project was to examine whether banks’ public-facing ESG messaging aligned with the positions they expressed in regulatory comment letters. To do this, I used R-based text analysis methods to study tone, language patterns, and recurring themes across bank documents.

The project focused on identifying how institutions discussed climate-related regulation and whether the language in those responses suggested support, hesitation, or resistance.

## Methods

This project used text analysis tools in R to process and evaluate public bank comment letters.

Key methods included:
- sentiment analysis
- bigram analysis
- text scraping / document collection
- thematic comparison across institutions
- structured synthesis in Excel

The bigram work helped identify common phrase patterns across letters, while the sentiment analysis helped capture tone and stance in how banks discussed climate-related policy proposals.

## Research Question

How do major banks communicate about climate-related regulation in public comment letters, and to what extent does that communication align with their broader ESG positioning and disclosures?

## Workflow

A general version of the workflow for this project was:

1. Collect and organize public bank comment letters and related text sources  
2. Clean and preprocess the text data in R  
3. Run sentiment analysis to evaluate tone and language patterns  
4. Use bigram analysis to identify recurring phrase structures and framing patterns  
5. Compare patterns across banks and summarize findings in a structured format  
6. Translate results into concise research summaries for faculty-led work

## Key Focus Areas

This project focused on questions such as:
- whether banks used consistently positive, neutral, or negative language when discussing climate-related regulation
- which phrases and framing patterns appeared most often across institutions
- whether banks’ regulatory comments appeared aligned with their public ESG commitments
- how text analysis could support broader policy and disclosure research
